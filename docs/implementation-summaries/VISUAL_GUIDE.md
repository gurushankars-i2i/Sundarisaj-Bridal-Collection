### Spacing Pattern
- **Sections:** 4rem vertical padding
- **Cards:** 2rem gap between items

- **Max Width:** 1400px containers

### Typography Scale
- **Main Titles:** 2.5rem, bold
- **Subtitles:** 1.1rem, light
- **Body:** 1rem, regular
- **Small:** 0.9rem-0.85rem

- **Standard:** 0.3s ease
- **Animations:** 0.5-0.6s ease-out
- **Quick:** 0.2s ease
